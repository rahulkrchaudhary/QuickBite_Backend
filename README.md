# QuickBite Backend 🍽️

This is the backend for **QuickBite**, built with **Spring Boot** and connected to **Supabase PostgreSQL**. It provides REST APIs for user management, restaurants, orders, payments, and more.

## 🛠️ Tech Stack

- **Spring Boot**
- **Spring Data JPA**
- **PostgreSQL (via Supabase)**
- **Docker**
- **Render** (Deployment)
- **JWT** (Authentication)
- **Spring Security**

## 🔐 Core Features

- User Registration & Login (JWT-based)
- Restaurant & Menu Management
- Cart & Order System
- Payment Integration (Stripe or Razorpay – optional)
- Admin panel APIs
- Real-time order status tracking (WebSockets – optional)
